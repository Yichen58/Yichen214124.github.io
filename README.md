# Airbnb Data Analysis Tutorial for Business Students

Welcome to your Python data analysis journey! This tutorial is designed specifically for first-year business students who want to learn Python through practical data analysis using real Airbnb data.

## 🎯 Learning Objectives

By the end of this tutorial, you will be able to:
- Understand basic Python data types and operations
- Read and explore CSV data files
- Clean and filter data for analysis
- Calculate basic statistics (mean, median, counts)
- Create simple data visualizations
- Extract business insights from data

## 📚 Tutorial Structure

### Module 1: Python Basics for Data Analysis
- `01_python_basics.py` - Variables, data types, and basic operations
- `02_working_with_lists.py` - Lists and basic data structures

### Module 2: Reading and Exploring Data
- `03_reading_data.py` - Loading CSV files with pandas
- `04_data_exploration.py` - Understanding your dataset

### Module 3: Data Cleaning and Analysis
- `05_data_cleaning.py` - Handling missing data and duplicates
- `06_basic_statistics.py` - Calculating means, medians, and counts

### Module 4: Data Visualization
- `07_simple_charts.py` - Creating bar charts and histograms
- `08_business_insights.py` - Extracting meaningful business insights

### Module 5: Practice Exercises
- `exercises/` - Practice problems with solutions

## 🗂️ Files in this Repository

- `data/airbnb_sample.csv` - Sample Airbnb dataset for practice
- `requirements.txt` - Python packages you'll need
- `solutions/` - Solution files for exercises

## 🚀 Getting Started

1. **Install Python** (version 3.7 or higher)
2. **Install required packages**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Start with Module 1** and work through each file in order
4. **Run each Python file** to see the examples in action

## 💡 Tips for Success

- Read the comments in each file carefully
- Try modifying the code to see what happens
- Complete the exercises to reinforce your learning
- Don't be afraid to experiment!

## 📊 About the Dataset

The sample Airbnb dataset contains information about:
- Property listings and prices
- Location data (neighborhoods, cities)
- Property types and amenities
- Host information
- Review scores and availability

This real-world data will help you understand how businesses use data analysis to make decisions.
